1 Запустить Main.java или FileManager.jar

2 Ввести любое число от 0 до 5.

3 Если ввели 0 - Программа завершит свою работау.

4 Если ввели 1 - Программа перейдет в режим создания файла.
    4.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите "E:/".
    4.2 Программа запросит имя файла. Для примера: введите "test.txt".
    4.3 Программа запросит ввести текст для файла. Для примера: введите "Hello".
    4.4 Программа создаст файл с введенным текстом в указанном месте и перейдет в режим ожидания следующей команды.

5 Если ввели 2 - Программа перейдет в режим удаления файла.
    5.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите "E:/".
    5.2 Программа запросит имя файла. Для примера: введите test.txt (этот файл должен существовать, иначе программа выведет ошибку о том, что такого файла нет).
    5.3 Программа удалит файл и перейдет в режим ожидания следующей команды.

6 Если ввели 3 - Программа перейдет в режим переименования файла.
    6.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите "E:/".
    6.2 Программа запросит имя файла. Для примера: введите test.txt (файл должен существовать).
    6.3 Программа запросит новое имя для файла. Для примера: введите supertext.txt.
    6.4 Программа переименует файл и перейдет в режим ожидания следующей команды.

7 Если ввели 4 - Программа перейдет в режим поиск слова в файле.
    7.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите E:/.
    7.2 Программа запросит имя файла. Для примера: введите supertext.txt (файл должен существовать).
    7.3 Программа запросит слово для поиска в файле.
        Для примера: введите "text"
        Предварительно в файл необходимо внести данные.
        Для примера: This new text
                     This new text2
                     This new text3
                     This new text4.
    7.4 Программа найдет номера строки и слова в файле и перейдет в режим ожидания следующей команды.

8 Если ввели 5 - Программа перейдет в режим замены слова в файле.
      8.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите E:/.
      8.2 Программа запросит имя файла. Для примера: введите supertext.txt (файл должен существовать).
      8.3 Программа запросит слово для замены. Для примера: введите "text".
      8.4 Программа запросит слово, на которое будет произведена замена. Для примера: введите "noText".
      8.5 Программа заменить все слова на новое, покажет результат и перейдет в режим ожидания следующей команды.
